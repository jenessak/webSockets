Print on both sides of a piece of paper (Duplex Printing)

Find out if your printer supports duplex printing

1. Click File > Print.
2. In the Printer list, select the printer you want to use, then click Printer Properties.
3. If you see options for "duplex printing," "print on both sides," or something similiar, your printer supports duplex printing.

Print using duplex support

1. Click File > Print.
2. In the Printer list, select the printer you want to use
3. In Settings, choose "Print on Both Sides - Flip sheets on short edge".

Print on a printer that does not support duplex printing

1. Click File > Print.
2. In the Printer list, select the printer you want to use
3. Print the first page by typing 1 in the "Pages" box, then click "Print".
4. When the first side is printed, flip the sheet over, then reinsert it into the printer. Print the other side by typing 2 into the "Pages" box.
